# AM1BaseFrame21LTS
フレームワークや共通コード、フェーズシステムなどの自家製汎用パッケージの開発リポジトリです。本リポジトリのサブフォルダーで複数のパッケージの開発と公開をしています。

まだ実装方針から研究しながら実装しているので安定したライブラリではありません。ドキュメントも整理中です。

## 対象バージョン
- 開発 Unity2021.3.16f1
- 対応 Unity2020LTS以降
